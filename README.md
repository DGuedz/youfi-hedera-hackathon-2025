# 💜 YouFi Protocol

> **Revolucionando Finanças Pessoais com Blockchain, IA e Gamificação**

**🏆 Projeto para o Hedera Hello Future: Origins Hackathon 2025**

## 🌟 Visão Geral

O YouFi Protocol é uma plataforma inovadora que combina **finanças pessoais**, **blockchain** e **inteligência artificial** para criar uma experiência gamificada e colaborativa de economia e investimento. Utilizando a rede Hedera Hashgraph, oferecemos uma solução completa para jovens brasileiros alcançarem suas metas financeiras.

## 🚀 Funcionalidades Principais

### 📱 **YouFi Wallet - App Mobile**

* **Metas Financeiras Gamificadas** com sistema de badges e recompensas
* **SBTs (Soul Bound Tokens)** para verificar conquistas na blockchain
* **IA Personalizada** para análise de gastos e recomendações inteligentes
* **Sistema de Apoio Coletivo** para metas compartilhadas
* **Integração Open Finance** via Pluggy para monitoramento em tempo real

### 🌐 **Plataforma Web**

* **Dashboard Completo** para gestão de metas e investimentos
* **DeFi Integration** com SaucerSwap e HBAR staking
* **Cross-Chain Bridges** para acesso a DeFi global
* **Analytics Avançados** com insights baseados em IA
* **Comunidade** para compartilhamento de metas e experiências

### 🔗 **Integração Blockchain (Hedera)**

* **HCS (Hedera Consensus Service)** para registro de transações
* **HBAR Staking** para geração de yield automático
* **Smart Contracts** para automação de metas e recompensas
* **SBT Minting** para certificação de conquistas financeiras
* **Interoperabilidade** com outras redes via bridges

## 🛠️ Stack Tecnológica

## 🚀 Como Começar

### **Para Usuários**

1. **Baixe o YouFi Wallet** (PWA disponível)
2. **Conecte sua conta bancária** via Pluggy
3. **Defina suas metas** financeiras
4. **Ganhe recompensas** por consistência
5. **Invite amigos** para apoio coletivo

### **Para Desenvolvedores**

```bash
# Clone o repositório
git clone https://github.com/youfi-protocol/youfi-app

# Instale dependências
npm install

# Configure variáveis de ambiente
cp .env.example .env.local

# Execute em desenvolvimento
npm run dev
```
